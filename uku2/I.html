Problem I: Tree-Light

Problem

あなたはツリーライトという木型の照明器具を買ってきた。

この照明器具にはn個の接点があり、0からn-1までの番号がついている。各接点は10段階の明るさを表現できる電球と、電球の状態を切り替えるためのダイアルで構成される。最初、すべての電球は明るさ0の状態である。
また、接点と接点の間にはケーブル線がある。すべての接点はケーブルによってツリー状に繋がっており、接点0を上にして吊り下げられる。ここで接点iから下方向に0本以上のケーブル線を介して、接点iと繋がる接点の集合を、接点iを根とした部分木と呼ぶ。

接点iのダイアルをプラス方向に|x|だけ回すと、接点iを根とした部分木に含まれる接点の電球の明るさが|x|だけ明るくなり、マイナス方向に回すと|x|だけ暗くなる。ただし、明るさ9の状態でプラス方向に1だけダイアルを回したときの明るさは明るさ0、明るさ0の状態でマイナス方向に1だけダイアルを回したときの明るさは明るさ9になる。

あなたはこの照明器具に対し、次のいずれかの行動をとる。

  • 接点rのダイアルをxだけ回す
  • 接点rを根とした部分木に含まれる明るさyの電球の数を数える

q回の行動が与えられるので、電球を数える行動が与えられたときにその時点での電球の数を出力せよ。

Input

入力は以下の形式で与えられる。

    n q
    u1 v1
    u2 v2
...
    un-1 vn-1
    t1 r1 s1
    t2 r2 s2
...
    tq rq sq

1行目にツリーライトを構成する接点の数n、あなたのとる行動の数qが与えられる。
続くn-1行にはケーブル線の情報が与えられる。i番目のケーブル線の情報は、ケーブル線iが接点uiを上にして、接点uiと接点viを繋ぐことを表す。
n+1行目以降のq行にはあなたのとる行動が与えられる。ti = 1なら接点riのダイアルを回す。このときx = siとする。ti = 2なら接点riを根とした部分木に含まれる電球の数を数える。このときy = siとする。

Constraints

  • 1 ≤ n ≤ 105
  • 1 ≤ q ≤ 105
  • 1 ≤ ui, vi, ri ≤ n-1 (ui != vi)
  • ti = 1のとき、-10 ≤ si ≤ 10
  • ti = 2のとき、0 ≤ si ≤ 9

Output

電球を数える行動が与えられたとき、その答えを出力する。

Sample Input 1

7 5
0 1
1 2
1 3
0 4
4 5
4 6
1 0 5
1 4 3
2 0 8
1 4 -3
2 0 5

Sample Output 1

3
7

Sample Input 2

7 5
0 1
1 2
1 3
0 4
4 5
4 6
1 0 5
2 0 5
1 1 -8
1 4 2
2 0 7

Sample Output 2

7
6
