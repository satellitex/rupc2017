Problem H: Zombie Hunter

Problem

がっちょ君は人気のゲーム、Zombie Hunterに熱中している。 Zombie Hunterの世界では頭、胴、腕、腰、足の５種類の防具があり、体の各部に防具を最大１つまで装備することができる。
それぞれの防具には２種類のポイントxi , yi があり、装備した防具のそれぞれのポイントの和がA以上、B以下になるとスキルが発動する。
そこで、がっちょ君の所有している防具の情報が与えられるので、がっちょ君がスキルを発動することができるかを調べて欲しい。

Input

入力は以下の形式で与えられる。

N A B
t1 x1 y1
t2 x2 y2
...
tN xN yN

入力はすべて整数で与えられる。
1行目にがっちょ君が所有している防具の数N, スキルが発動するために必要なポイントの下限A、上限Bが空白区切りで与えられる。
2行目から2+N行目はがっちょ君の所有している防具の種類を表す数字ti, 2つスキルのポイントの値xi, yi がそれぞれ空白区切りで与えらえる。

tiの値はそれぞれ
1のとき頭の防具、
2のとき胴の防具、
3のとき腰の防具、
4のとき腕の防具、
5のとき足の防具、
であることを表す。

Constraints

入力は以下の条件を満たす。

  • 1 ≤ N ≤ 500
  • 1 ≤ A ≤ B ≤ 1012
  • 1 ≤ ti ≤ 5
  • 1 ≤ xi,yi ≤ 1012

Output

がっちょ君が2つのスキルを同時発動することができる場合は”Yes”、できない場合は”No”を1行に出力せよ。

Sample Input 1

7 10 11
1 1 0
1 0 1
2 2 2
3 3 5
4 1 3
5 1 3
5 5 -1

Sample Output 1

Yes


がっちょ君が
頭に(0,1)
胴に(2,2)
腰に(3,5)
腕に(1,3)
足に(5,-1)
の防具を装備すると
1つめのスキルポイントの和が 0+2+3+1+5 = 11
2つめのスキルポイントの和が 1+2+5+3-1 = 10
になり両方の値がA以上でかつB以下なのでスキルを発動させることができる。

Sample Input 2

7 10 10
1 1 0
1 0 1
2 2 2
3 3 5
4 1 3
5 1 3
5 5 -1

Sample Output 2

No
